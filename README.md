# Leon

A small, responsive agency landing page built with HTML and CSS.

🌐 **[Live Preview](https://leon-static.vercel.app/)**

## About

This project contains a one-page website template for a creative agency. It includes:

- Landing section
- Services overview
- Portfolio section
- About and contact sections

## Files

- `index.html` — main site markup
- `css/master.css` — primary stylesheet
- `css/Normalize.css` — normalize styles
- `css/all.min.css` — Font Awesome icons
- `Images/` — site images and logo
